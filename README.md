# 介绍

从 2026.7.1 开始搞的玩意  

使用 Rust 语言、Direct3D11（暂时）的游戏引擎（未完工）  

目前只完成了键鼠输入、基础的2D图形、相机变化、文字（含Emoji）渲染，且仍有许多部分需要完善  

> **作者的话**：虽然“引擎”尚未成熟，但起码能写出像[`app_fish`](app_fish/src/app.rs)那样的小游戏了，证明这个项目还是有点价值的（  

## 素材声明

`app_fish`与`app_sethsweeper`的音频素材来自游戏**Deltarune**，`app_sethsweeper`的图片素材来自网络，仅用于暂时性的学习交流。
